## Technology Used
For this project Unity version 6000.0.39f1 was selected for its improved compatability with Android platforms and inclusion of the latest Unity features, Git and GitLFS was our preferred version control software, and used GitHub Desktop as our primary way to interface with this command line tool, and to work collabouratively on the game.

### Programming
In programming the game, Visual Studio (2022) was selected for both its compatability with C# and Unity and was the main IDE for the project, whilst Visual Studio Code was used for changes to the README, as well as for a smaller portion of the C# programming.
